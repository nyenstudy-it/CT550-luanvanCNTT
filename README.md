## Luan van - Sen Hong OCOP

Project Laravel cho hệ thống SEN HỒNG OCOP.

### Yêu cầu môi trường
- PHP (khuyến nghị theo Laravel 11)
- Composer
- MySQL/MariaDB (XAMPP)

### Thiết lập nhanh
```bash
composer install
copy .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
